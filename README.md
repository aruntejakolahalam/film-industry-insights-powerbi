# Film Industry Insights – Trends and Performance Analysis

## Overview
This project analyses trends and performance in the film industry using data from the IMDb
movie dataset. The objective is to explore revenue patterns, audience engagement, and key
contributors to film success through interactive Power BI visualisations.

## Key Analysis Areas
- Revenue performance by year, genre, and rating
- Audience engagement trends over time
- Identification of critically successful films
- Director performance based on rankings and outcomes

## Tools Used
- Microsoft Power BI
- Data modelling and transformation
- Interactive dashboards and visual analytics

## Files in This Repository
- Film_Industry_Insights_IMDb_PowerBI_Dashboard.pbix – Power BI dashboard file  
- Film_Industry_Insights_IMDb_Report.pdf – Analytical report and insights  

## Key Insights
- Certain genres consistently generate higher average revenue.
- Films with higher audience ratings tend to achieve stronger commercial performance.
- A small group of directors contribute disproportionately to critically successful films.
- Revenue peaks are observed in specific release periods.
